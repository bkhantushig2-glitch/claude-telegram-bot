# Format Guide by Subject Area

## Science Topics
- Include methodology or experiment details when relevant
- Use precise units and measurements
- Note whether findings are established consensus or recent research

## History Topics
- Include a brief timeline if the topic spans multiple events
- Use specific dates and names
- Note primary vs. secondary source distinctions

## Current Events
- Date-stamp all facts (e.g., "as of April 2026")
- Clearly separate confirmed facts from analysis/opinion
- Include multiple perspectives when the topic is political

## Technical Topics
- Include version numbers, release dates for software/tools
- Link to official documentation when available
- Note compatibility or requirements if relevant

## General Tips
- When the user asks about a place (city, country), include practical info like population, geography, and notable facts
- When the user asks about a person, lead with their role and key achievements
- When the user asks "how to" do something, structure the response as numbered steps
